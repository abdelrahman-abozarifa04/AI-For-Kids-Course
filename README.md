**# AI Track for Teens (Ages 15-18) - 2-Year Program**

## **Program Overview**
This AI program helps teenagers (ages 15-18) learn about artificial intelligence, machine learning, and how AI is used in the real world. The course is split into four terms over two years. It starts with the basics and moves to advanced AI topics through fun, hands-on projects, coding exercises, and interactive lessons. Each term includes theory, coding practice, and a big project at the end to test what students have learned.

## **ðŸ“Œ Term 2: Introduction to Machine Learning**
### **Goal:** Learn how AI can learn from data and make predictions.

### **Topics:**
1. **What is Machine Learning (ML)?**
   - How ML helps computers learn from experience
   - Types of ML: Supervised vs. Unsupervised Learning
   - How AI trains on data and improves over time

2. **Simple AI Models**
   - Predicting numbers with Linear Regression
   - Sorting things into categories (K-Nearest Neighbors, Decision Trees, Random Forest)
   - Grouping similar things with Clustering (K-Means)
   - Training ML models using Scikit-Learn

3. **Neural Networks & Deep Learning**
   - How AI learns using layers of neurons (like the human brain!)
   - Basics of Neural Networks: Perceptrons and Activation Functions
   - Teaching AI with Backpropagation (fixing mistakes as it learns)
   - Making a simple AI model with TensorFlow/Keras

4. **AI in Real Life**

   - AI in robots and self-driving cars
